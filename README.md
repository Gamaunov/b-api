## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit test
$ pnpm run test

# e2e test
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
